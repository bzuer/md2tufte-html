### Subtítulo Nível 3

#### Lista não ordenada
- Item principal 1
- Item principal 2
  - Subitem 2.1
  - Subitem 2.2
- Item principal 3[^1]

#### Lista ordenada
1. Primeiro passo
2. Segundo passo
3. Terceiro passo
4. 

## Tabela Exemplo

| Coluna 1       | Coluna 2          | Coluna 3     |
|----------------|-------------------|--------------|
| Linha 1, Cel 1 | Dados importantes | 123         |
| Linha 2, Cel 1 | Mais dados        | 456         |
| Linha 3, Cel 1 | Exemplo final     | 789 |

Veja:

> Esta é uma citação em bloco (blockquote). Útil para destacar trechos importantes ou citações de fontes externas. 


## Bloco de Código

Aqui está um exemplo de código em Python:

```python
def saudacao(nome):
    """Função simples que retorna uma saudação."""
    return f"Olá, {nome}!"

print(saudacao("Mundo"))
```

E um exemplo de código inline: `print("Hello World")`.
